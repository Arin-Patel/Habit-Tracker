# Habit Tracker

A simple habit tracker web application built using HTML, CSS, JavaScript, and Firebase.

## Features
- Add and manage daily habits
- Track completion status
- Persistent data using Firebase
- Daily reminder by sending E-mail

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase

## Security Note
Firebase configuration values are excluded from the repository for security reasons.
